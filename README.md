# Chemistry_AR

---- Intro ----

This application has been implemented using behaviourist and gamification learning theories.

The goal is to learn the names of chemical elements and their chemical type.

Allow all the chemical elements to spawn in, then tap the screen to place three collection baskets.
Once placed, use touch/mouse input to grab and drag elements into the baskets. 

---- Instructions ----
1. Navigate to scenes folder
2. Open SampleScene
3. Press play
4. Use touch/mouse to interact
